Corel(R) Painter(TM) Essentials 3 發行說明

歡迎使用 Corel(R) Painter(TM) Essentials 3 － 輕鬆將您的照片轉換成寫真的圖畫！

此文檔涵蓋有關 Corel Painter Essentials 3 的說明資訊。

基本系統需求

Macintosh® 系統：

• Mac OS® X (10.2.8 版或有最新 OS 更新的較高版本)
• Power Macintosh® G3, 500 MHz 以上
• 256 MB 的 RAM
• 滑鼠或繪圖板
• 24 位元色彩顯示
• 1,024 × 768 或更大顯示器解析度
• 150 MB 磁碟安裝空間

Windows® 系統：

• Windows 2000, Windows XP 或 Microsoft Vista (安裝有最新的更新程式)
• Pentium® II, 500 MHz 以上
• 256 MB 的 RAM
• 滑鼠或繪圖板
• 24 位元色彩顯示
• 1,024 × 768 或更大顯示器解析度
• 150 MB 磁碟安裝空間

Corel Painter Essentials 3 [說明] 系統需要網頁瀏覽器，透過使用 Macintosh 上的 Safari(TM) 網頁瀏覽器和 Windows 上的 Microsoft(R) Internet Explorer 能夠獲得最佳的效果。

安裝

Corel Painter 較舊的版本

Corel 不建議將 Corel Painter Essentials 3 安裝於舊版之上。請安裝於新的檔案夾。

安裝 Corel Painter Essentials 3 前須知

在安裝 Corel Painter Essentials 3 之前，您必須關閉所有其他的應用程式，包括病毒防護軟體在內。如果其他應用程式正在執行，可能會妨礙安裝的過程。在 Windows 上，開啟的應用程式顯示在工作列上。在 Mac OS 系統上，開啟的應用程式圖像顯示在 Dock 上，其下有一個黑色的三角形。

在 Macintosh 系統上安裝須知

Macintosh 電腦上的 UNIX(R) 分割區不能安裝 Corel Painter Essentials 3。

序號

您必須有 Corel Painter Essentials 3 的序號才能安裝 Corel Painter Essentials 3。試用版本除外。請將此序號妥善保管，以便在重新安裝 Corel Painter Essentials 3 或致電 Corel 支援服務時使用。若啟動過試用版，您也必須記下您的啟動代碼。

試用版本

在非管理員帳戶上啟動試用版本之前，必須先在管理員帳戶上啟動過 Corel Painter Essentials 3 應用程式。

 

Corel Painter Essentials 3 技術資訊

若要將使用者設定重設到預設出廠設定，請在啟動應用程式時按住 Shift 鍵。便會出現提示窗口，請您確認是否要重設到預設出廠設定。按一下 [是]。請注意以上的操作將使您以前自訂的設定流失。

「歡迎」螢幕

有些「歡迎」螢幕上的藝術家網站上會顯示日文。如果您選擇在非日文版本的 Windows XP 或 Windows 2000 操作系統上瀏覽這些網站，但是還未在系統上安裝日文字元組，您可能會被提示要安裝日文字元。

「歡迎」螢幕上介紹許多藝術家的連結在測試時期可連上，不過之後有些網站可能已經不運行，而這不在我們的控制範圍內。

數位水彩

- 不是所有的效果都設計成適合用在數位水彩。若是將這些效果用在數位水彩上，可能對數位水彩的筆刷筆觸不會造成任何的改變。
- 在儲存成 PSD 檔時，Corel Painter 將數位水彩變乾，並將內部合成方式設定成「變暗」。這個方式比較適合用在淺色背景的影像，不過對於深色背景的影像，您可能要先將 Adobe(R) Photoshop(R) 中的混色模式改成「色彩增值」，可能會達到較好的效果。

檔案篩選器：

- Corel Painter Essentials 3不支援有圖層效果的 Adobe(R) Photoshop(R) 檔案。如果您的 Adobe Photoshop 檔案有圖層效果，建議您先在畫布上落下圖層效果，然後在 Corel Painter Essentials 3 中開啟此檔案。

- Corel Painter Essentials 3 不支援 LZW 壓縮的 TIFF 檔案。

文字

- Corel Painter Essentials 3 僅支援 Mac OS X 中的 Open Type 字型。

使用者介面

- Corel Painter Essentials 3 能夠在 Windows 2000 及 Windows XP 中的小字體顯示下獲得最佳效果。

全球化 (僅限 Windows 系統)

Corel Painter Essentials 3 不是一個 Unicode 的應用程式。如果 Corel Painter 是安裝在有多語言軟件包的英文版 Windows 作業系統，而您想要改變作業系統的語言，您必須先將系統的「地區選項」改成該語言的地區，應用程式才會相繼正確運作。這個步驟很重要，尤其是如果您要使用的語言包含雙字節文字 (如日文)。您必須依照以下方式將「地區選項」中的地區改成該語言的地區：

Windows XP：

1. 按一下「開始」、「控制台」、「地區和語言選項」。

2. 選擇「進階」標籤。

3. 從「非 Unicode 程式的語言」下拉式清單中選擇「日文」。

Windows 2000：

1. 按一下「開始」、「控制台」、「地區和語言選項」。

2. 按一下「設定預設值」按鈕。

3. 從下拉式清單中選擇「日文」。

註：

- 如果您是在本機語言的作業系統下操作，例如 Windows 2000 日文版，您不用額外改設地區選項，因為該語言的作業系統會自動將設定預設的語言設定成該系統的本機語言。
- 在日文版作業系統上使用 Corel Painter Essentials 3 英文版時，不支援拉丁語字母的擴展字元。

試用版、教育版、和 OEM 版本

並非所有 Painter Essentials 元件皆包含於軟體的試用版、教育版、和 OEM 版本中。不包含的元件可能包括了 Corel 應用程式、lynda.com 訓練、功能、協力商公用程式和附贈的內容文檔。然而，某些上列的元件可能還是會在應用程式中或文件說明中被提及。
